Overview

Mâlie renders a live wallpaper on your selected monitor (primary by default) using WeatherAPI weather data, a local Three.js procedural scene pipeline, and optional Meshy/LatLng integrations for POI model generation.

The animated background mode uses a Codex-inspired multi-layer motion shader blended with your chosen wallpaper color.

Requirements

  • Windows 11, .NET 8 SDK, and WebView2 runtime.
  • Node.js + npm for Angular/PrimeNG UI build.
  • Required API keys: WeatherAPI, Meshy, and LatLng server key.

Onboarding Steps

  1. Set location (city, address, or ZIP), or use current location.
  2. Provide WeatherAPI, Meshy, and LatLng keys.
  3. Load POIs and download/import at least one Meshy GLB model.
  4. Finish onboarding to connect the wallpaper renderer.

WeatherAPI Key

  • Sign up: weatherapi.com/signup.aspx
  • Docs: weatherapi.com/docs
  • Paste the key in Settings or onboarding API key step.

Build EXE + MSI

Generate both installers with one command:

.\build-installers.ps1 -Runtime win-x64 -Version 1.0.0 -AcceptWixEula
  • EXE installer uses Inno Setup (`ISCC.exe`).
  • MSI installer uses WiX CLI (`wix.exe`, v7 recommended).
  • Optional: -SkipUiBuild, -SkipClean, -AcceptWixEula.
  • OSMF/EULA docs: docs.firegiant.com/wix/osmf
  • One-time WiX EULA acceptance: wix eula accept wix7.
  • Direct WiX build form: wix build -acceptEula wix7 ....
  • Sun icon is used for setup EXE branding and installed Start Menu/Desktop shortcuts.
  • Outputs are written to installer\dist\.

Runtime Behavior

  • App minimizes to tray. To fully quit, right-click tray icon and choose Exit.
  • Settings close does not terminate process.
  • Logs are written to %LOCALAPPDATA%\Malie\logs\app-debug.log.
